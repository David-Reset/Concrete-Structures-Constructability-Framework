Framework Doctor
Read-only health check for data.json and the images folder — runs entirely in your browser
What this does
Load your data.json and (optionally) drop in your images folder. The Doctor cross-checks the two and reports problems: images referenced but missing from the folder, files in the folder nothing points to, broken cross-references, entries that look ready but are still drafts, and internal notes that ship in the published file. It never changes anything — it only reads and reports. The quickest way to get the folder: from the GitHub repo, use Code → Download ZIP, unzip, then either pick the images folder with the button below, or open it and drag the image files in.
